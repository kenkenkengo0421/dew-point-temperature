
# [露点温度](https://ja.wikipedia.org/wiki/露点温度)


## [演算ツールURL](https://kenkenkengo0421.github.io/dew-point-temperature/)

### マグヌス（Magnus）式

>飽和水蒸気圧：es = 6.112 * exp(17.67*T/(T+243.5))

>実際の水蒸気圧：e = es * RH/100

>露点：Td = 243.5*ln(e/6.112) / (17.67 - ln(e/6.112))

1結果 := 温度 + 243.5<br>
2結果 := 17.67 * 温度<br>
3結果 := 2結果 / 1結果<br>
4結果 := EXP (3結果)<br>
5結果 := 6.112 * 4結果<br>
6結果 := 5結果 * 湿度 / 100.0<br>
7結果 := 6結果 / 6.112<br>
8結果 := LN (7結果)<br>
9結果 := 243.5 * 8結果<br>
10結果 := 17.67 - 8結果<br>
<br>
露点温度 := 9結果 / 10結果<br>
